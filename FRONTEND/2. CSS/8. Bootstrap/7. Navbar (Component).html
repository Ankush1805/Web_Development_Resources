  • navbar: Base class for the navbar.
  • navbar-expand-lg: Expands the navbar on large screens and up.
  • navbar-light bg-light: Applies a light background color and text color to the navbar.
  • navbar-dark bg-dark: Applies a dark background color and light text color to the navbar.
  • navbar-brand: Defines the brand logo or name.
  • navbar-toggler: Defines the toggle button for collapsing the navbar on smaller screens.
  • navbar-toggler-icon: Adds a default icon to the navbar toggler button.
  • collapse navbar-collapse: Defines the collapsible part of the navbar.
  • navbar-nav: Contains the navigation items.
  • nav-item: Defines each item in the navbar.
  • nav-link: Defines the links inside the navbar items.
  • active: Highlights the current page's link.
  • dropdown: Defines a dropdown menu within the navbar.
  • dropdown-menu: Styles the dropdown menu container.
  • dropdown-item: Defines each item in the dropdown menu.
  • dropdown-toggle: Indicates that the dropdown is toggleable.
  • nav-item dropdown: Combines nav item with dropdown for a dropdown link.
  • form-inline: Styles forms within the navbar to be inline.
  • navbar-text: Adds custom text within the navbar.
  • navbar-collapse: Provides responsive collapsing functionality.
  • navbar-fixed-top: Fixes the navbar to the top of the viewport (legacy, use fixed-top instead).
  • navbar-fixed-bottom: Fixes the navbar to the bottom of the viewport (legacy, use fixed-bottom instead).
  • fixed-top: Fixes the navbar to the top of the viewport.
  • fixed-bottom: Fixes the navbar to the bottom of the viewport.
  • sticky-top: Sticks the navbar to the top of the viewport when scrolling.
